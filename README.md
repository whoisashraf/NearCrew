## Usage

This project uses JSON-Server for a mock backend.

### Install Dependencies

```fish
bun i
```

### Run JSON Server

The server will run on http://localhost:8080

```fish
bun run server
```

### Run Vite Frontend

React will run on http://localhost:5000

```fish
bun run dev
```

### Build for Production

```fish
bun run build
```

### Preview Production Build

```fish
bun run preview
```
